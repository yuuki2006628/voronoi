webサイト上に設置するには、

```
<canvas id="canvas"></canvas>
```

を任意のbodyタグ内に設置してください。
また、サイズは一番上に描いてあるコード

```
var width = window.innerWidth,
    height = window.innerHeight,
```

のwindow.innerWidthとwindow.innerHeightを任意の数字に変えて利用してください。
